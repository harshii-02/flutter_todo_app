# Flutter To-Do App ✅

A minimal to-do list app built using Flutter. Allows you to add, delete, and check off tasks with local data persistence.

## 🔥 Features

- Add new tasks
- Delete tasks with swipe
- Check/uncheck tasks
- Tasks saved locally using `shared_preferences`
